# Subarray Division
### Program to divide a list into 3 parts so that difference between sum of each group is minimum
**Usage:** java -jar exercise-0.1.jar <list of integers>  
**Sample:** java -jar exercise-0.1.jar 7 1 2 -5 8 4 3  

Result:  
Part 1: [7, 1, 2], Sum: 10  
Part 2: [-5, 8, 4], Sum: 7  
Part 3: [3], Sum: 3  
Difference: |10 - 7 - 3| = 0
